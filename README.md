# Tentang Perusahaan
PT. OTICS Indonesia merupakan pabrik pertama OTICS Corp yang berada di luar negeri. Kami memproduksi Mesin, Sasis, dan Suku Cadang Presisi. Perkembangan kami berdasarkan landasan teknologi. Kami selalu berusaha untuk menyediakan produk berkualitas tinggi dan sangat andal melalui teknologi canggih untuk memuaskan pelanggan kami.
# Deskripsi Produk
Untuk sepenuhnya mencerminkan teknologi baru dalam produksi, kami selalu memanfaatkan fasilitas terbaru sehingga kami dapat memenuhi kebutuhan pelanggan dan bertujuan untuk produksi yang ideal. Sekaligus Mengkristalkan teknologi dalam produksi suku cadang presisi, OTICS bertujuan untuk memperbaiki masa depan umat manusia dan mobil.
# Visi
Visi perusahaan adalah Perusahaan dengan Kualitas Part Otomotif Terbaik di ASIA
Mewakili  komitmen OTICS sebagai perusahaan part otomotif  untuk mengembangkan
proses manufaktur terbaik dan menghasilkan produk berkualitas terbaik di Asia
# Misi
Misi perusahaan adalah Memenuhi kebutuhan part otomotif customer melalui
kerja keras dengan keselarasan ide-ide inovatif dan orisinil, peningkatan kualitas, penurunan biaya,
memajukan perusahaan dan mensejahterakan karyawannya, serta memberikan kontribusi pada masyarakat
